Python Code

Gambar sebelah kiri menampilkan kode program Python yang digunakan untuk mengambil jadwal adzan. Program ini menggunakan library requests untuk mengakses API jadwal sholat dan tabulate untuk mengolah serta menyiapkan data agar bisa ditampilkan dalam bentuk tabel. Kode disusun secara terstruktur mulai dari input nama kota, pengambilan data dari API, hingga pengolahan data waktu sholat.

Gambar sebelah kanan menampilkan proses eksekusi kode Python beserta hasil outputnya di terminal. Hasil ini menunjukkan bahwa kode Python berhasil mengambil dan memproses data dari API dengan baik.
